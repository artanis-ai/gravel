Gravel is an open-source library that mounts an admin dashboard inside your AI app. Domain experts — clinicians, lawyers, teachers, recruiters, accountants, whoever — review output, manage prompts, and run evals, **without ever touching your codebase**.

Where Langfuse / LangSmith pitch dashboards to engineers, Gravel ships a dashboard for the people whose job *isn't* engineering — the ones who actually know what good looks like in your domain.

<Card title="Status: v0.5 (active development)" icon="circle-info">
  Gravel is being built in the open. The install wizard, dashboard, tracing (OpenAI / Anthropic / LangChain / Vercel AI / fetch), prompt PR submission via the GitHub App, and live migrations all ship today. See [STATUS.md](https://github.com/artanis-ai/gravel/blob/main/STATUS.md) for the up-to-date progress.
</Card>

## What you get

<CardGroup cols={2}>
  <Card title="One-command install" icon="terminal">
    `npx @artanis-ai/gravel init` (or `uvx artanis-gravel init`) detects your framework, adds the SDK to your project deps, runs OAuth, AST-edits your server entry, runs migrations, and sends a test trace before exit. Sentry-style. The same Go binary drives both stacks.
  </Card>
  <Card title="Runs in your infra" icon="shield">
    Traces, prompts, feedback, and datasets all live in your own database. Apache 2.0 lib. Only paid eval calls cross to our servers.
  </Card>
  <Card title="Git-first prompts" icon="code-branch">
    Prompts live where they live in your repo (files or string literals). Edits become PRs via the shared GitHub App.
  </Card>
  <Card title="Free for the basics" icon="check">
    Prompt management, tracing, feedback, datasets — free, unlimited seats. Pay only for our judges (trace evals, live evals, unlimited Mallet analysis).
  </Card>
</CardGroup>
