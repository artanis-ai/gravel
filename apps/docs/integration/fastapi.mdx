```bash
uvx artanis-gravel init    # or pipx run artanis-gravel init
```

The wizard generates:

```py gravel_route.py
from artanis_gravel.fastapi import create_gravel_router
from gravel_config import config

router = create_gravel_router(config=config)
```

And tells you to add to your `main.py`:

```py main.py
import artanis_gravel.auto  # boot tracing

from fastapi import FastAPI
from gravel_route import router
from gravel_config import config

app = FastAPI()
app.include_router(router, prefix=config.mount_path)
```

## Auth callback

```py gravel_config.py
from artanis_gravel import GravelConfig, GravelUser
from your_app.auth import get_current_user

async def get_user(req):
    user = await get_current_user(req)
    if not user:
        return None
    return GravelUser(
        id=str(user.id),
        first_name=user.first_name,
        role='admin' if user.is_admin else 'user',
    )

config = GravelConfig(
    mount_path='/admin/ai',
    database={'url': os.environ['DATABASE_URL']},
    auth={'get_user': get_user},
)
```
