## App Router

`npx @artanis-ai/gravel init` writes `app/admin/ai/[[...slug]]/route.ts`:

```ts app/admin/ai/[[...slug]]/route.ts
import { createGravelHandler } from '@artanis-ai/gravel/next'
import { config } from '@/gravel.config'

const handler = createGravelHandler({ config })

export const GET = handler
export const POST = handler
export const PUT = handler
export const DELETE = handler
```

## Pages Router

`pages/admin/ai/[[...slug]].ts`:

```ts pages/admin/ai/[[...slug]].ts
import { createGravelHandler } from '@artanis-ai/gravel/next-pages'
import { config } from '@/gravel.config'

export default createGravelHandler({ config })
```

## Tracing

At the very top of your server entry (e.g. `instrumentation.ts` or `app/layout.tsx`):

```ts
import '@artanis-ai/gravel/auto'
```

The wizard places this. It auto-patches OpenAI, Anthropic, Langchain, and Vercel AI SDK calls when those packages are installed. Disable with `GRAVEL_TRACING_DISABLED=1` if needed.

## Auth

The wizard pre-fills `getUser` for Clerk, NextAuth, Better Auth, Lucia, and Auth0. For everything else, fall back to default-password mode. See [/reference/auth](/reference/auth).
