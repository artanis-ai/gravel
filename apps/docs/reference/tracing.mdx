## Activation

Single import at the top of your server entry:

```ts
import '@artanis-ai/gravel/auto'
```

```py
import artanis_gravel.auto  # noqa: F401
```

The wizard places this. Default = enabled. Disable with
`GRAVEL_TRACING_DISABLED=1` if you want a black box.

## Coverage

| Provider              | TS package                   | Python package          |
| --------------------- | ---------------------------- | ----------------------- |
| OpenAI                | `openai`                     | `openai`                |
| Anthropic             | `@anthropic-ai/sdk`          | `anthropic`             |
| LangChain             | `langchain` / `@langchain/*` | `langchain`             |
| Vercel AI SDK         | `ai`                         | (TS only — no Py port)  |
| Raw fetch / HTTP      | `globalThis.fetch`           | `httpx`, `requests`, `aiohttp`, stdlib `urllib` |

Each patch is lazy: missing client library no-ops at import. Streaming
responses are traced without consuming the user's iterator (TS uses
`Symbol.asyncIterator` tee; Python wraps `__iter__` / `__aiter__`
similarly). Raw-`fetch` / `httpx` / `requests` / `aiohttp` / `urllib`
detection is path-based against OpenAI- and Anthropic-shaped URLs
(`/chat/completions`, `/responses`, `/embeddings`, `/v1/messages`), so
OpenAI-compatible proxies (Azure, vLLM, mock servers) are picked up
without configuration.

## Custom metadata

```ts
import { withGravelMetadata } from '@artanis-ai/gravel'

await withGravelMetadata(
  { userId: req.user.id, plan: 'pro', sessionId: 'sess_abc' },
  async () => openai.chat.completions.create({ /* ... */ }),
)
```

Metadata lands on the captured sample's `metadata` jsonb column
(`gravel_samples.metadata`) and shows up as filterable columns in the
dashboard.

## Async-context propagation

For request-scoped metadata that flows through every traced call:

```ts
// Express
app.use((req, res, next) => {
  gravelContext.run({ userId: req.user.id, route: req.path }, next)
})
```

```py
# FastAPI
from artanis_gravel.tracing import with_gravel_metadata

@app.middleware('http')
async def gravel_metadata(request, call_next):
    with with_gravel_metadata({'user_id': request.state.user.id}):
        return await call_next(request)
```

## Disable per-call

```ts
import { withTracingDisabled } from '@artanis-ai/gravel'

await withTracingDisabled(async () => {
  // not traced — useful for internal evals you don't want recursively traced
  await openai.chat.completions.create({ /* ... */ })
})
```
